# Voice-Enabled Task Tracker

## 1. Project Setup

### a. Prerequisites
- Node.js version: `vXX.X.X`
- MongoDB version: `X.X`
- AI API Keys: (e.g., OpenAI API key)
- Email configuration: SMTP credentials or third-party email service

### b. Installation Steps

#### Frontend
```bash
cd frontend
npm install
```

#### Backend
```bash
cd backend
npm install
```

#### c. Environment Variables
```bash
 Backend
PORT=4500
mongoURL=mongodb+srv://raja:*****@cluster0.vqouc0m.mongodb.net/speakTask?retryWrites=true&w=majority
OPENAI_API_KEY=**********************************************************************
```

#### d. Running Locally
```bash 
- Backend
cd backend
npm run dev

Runs backend on http://localhost:4500

- Frontend
cd frontend
npm start

Runs frontend on http://localhost:3000
```
